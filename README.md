## Install dependencies :

~~~bash
npm install -g babel-cli
gem install watch
~~~

## **Watch** and **build** the sources :

~~~bash
npm start
~~~

## Only Build the sources :

~~~bash
npm build_css
npm build_js
npm build
~~~
